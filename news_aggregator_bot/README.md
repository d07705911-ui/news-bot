# Новостной Агрегатор - Telegram Бот

Бесплатный агрегатор новостей с автоматической рассылкой.

## Функции
- 📰 Новости с Хабр, Лента.ру, N+1
- 🔥 Топ-5 новостей
- 📂 Категории
- ⭐ Избранное
- 📨 Автоматическая рассылка
- 🗑️ Автоудаление сообщений

## Команды
- /start - Главное меню
- /news - Все новости
- /top - Топ-5
- /subscribe - Подписка на рассылку
- /unsubscribe - Отписка

## Рассылка
- 🌅 Утренняя (9:00)
- 🌙 Вечерняя (18:00)
- ⚡ Новые новости (каждые 15 мин)

## Запуск локально
```bash
pip install -r requirements.txt
python bot.py
```

## Хостинг
Развёрнут на Render.com (бесплатный план)
